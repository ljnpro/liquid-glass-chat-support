Highlights

The short version

The mobile page leads with the parts people are most likely to look up on a phone.

Model and routing control

GlassGPT keeps GPT-5.4 and GPT-5.4 Pro model selection explicit, and can follow either direct OpenAI routing or a configured gateway path.

Generated file workflows

Code interpreter outputs show up as tappable links in chat, with native handling for images and PDFs instead of a dead-end download flow.

Native viewers

PNG, JPG, and JPEG files use the in-app image viewer, PDFs use a dedicated PDF viewer, and other generated files go straight to the iOS share sheet.

On-device caches

Generated images and generated documents are cached separately on device, with independent 250 MB limits and manual clear controls in Settings.

FAQ

Quick answers

The mobile page keeps these shorter and easier to scan.

How do I get an OpenAI API key?

Go to platform.openai.com/api-keys, create a key, enable billing on your OpenAI account, and paste the key into the Settings tab.

What models are supported?

GlassGPT supports GPT-5.4 and GPT-5.4 Pro, with explicit reasoning effort controls instead of a single hidden default.

How are generated files handled?

Images open in the image viewer, PDFs open in the PDF viewer, and everything else uses the system share sheet so you can pass the file to another app.

Are API keys and history stored on device?

Yes. Keys stay in on-device secure storage and conversations stay local to the device. Requests follow the configured API route without turning GlassGPT into its own analytics backend.
